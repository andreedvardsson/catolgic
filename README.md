# catolgic

A lightweight "OMFGCATS" static site built with plain HTML, CSS, and JavaScript.

## Run locally

```bash
cd /home/andre/catolgic
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Files

- `index.html`: page structure and audio element
- `cats.css`: visuals, layout, cat animation styles
- `js/wat.js`: runtime timer and cat spawning logic

## Optional audio

Add a file named `cats.mp3` in the repo root to play looping sound after pressing `Start Cats`.
